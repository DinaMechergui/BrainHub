 ##  Description du projet

Le site BrainHub est composé de plusieurs sections principales :

- **Organisation de la révision** : pages permettant de planifier, noter et suivre les révisions.  
- **Gestion du temps** : outils permettant à l’utilisateur de structurer son planning.  
- **Ressources variées** : pages offrant différents types de contenus utiles pour les études.  
- **Interface moderne et responsive** basée sur TailwindCSS.  

Le projet utilise uniquement **HTML5**, **CSS (Tailwind)** et **JavaScript**.
